# Imágenes

Coloca aquí los siguientes archivos:

- `logo.png` - Logo para modo claro
- `logo2.png` - Logo para modo oscuro

Puedes usar imágenes placeholder o crear tus propios logos. Las imágenes deben ser transparentes (PNG) y tener un tamaño apropiado para el header.

## Placeholder

Si no tienes logos, puedes usar imágenes placeholder o crear logos simples con herramientas online.
